# Watch Party Manager

Watch Party Manager is a self-hosted Discord bot for managing recurring watch parties.

It helps communities manage suggestions, voting, scheduling, Discord Events, watch history, and statistics while allowing each community to keep its own traditions.

---

## Project Status

**Current Phase:** Software Specification Review

The Version 1 specification has been drafted and is currently undergoing editorial review prior to implementation.

Development of the Python application will begin after the specification has been finalized.

---

## Planned Features

- Watch Item suggestions
- Rotation management
- Adaptive Balanced Pull
- Blind voting
- Discord Event creation
- Recurring Event Series
- Watch history
- Statistics
- Import and export
- Backup and restore
- Multi-server support
